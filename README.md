# How to use:

Simply go to `about:profiles` in your Floorp browser, then open folder for Root Directory.

Find a folder named `chrome`

Replace/Add this `userChrome.css` file in.

Restart Browser. (Either end task or from the restart option 

# Purpose:

I hated how the Floorp sidebar displayed the web panels you'd open. 

I wanted to have it overlay on top of my current active tab like Opera-GX, instead it tiled it next to one another.

This css file fixes that and brings back the Opera-GX like sidebar. (As long as you intend to use it for the openable mini webpages e.g Discord, Whatsapp, Insta ++)
